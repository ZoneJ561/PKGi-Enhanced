# PKGi Enhanced v1.1.44

## Changes

* Added installation of local ZIP files already stored on the Memory Stick.
* Added byte-based extraction progress for clearer feedback on large archives.
* Improved nested ZIP handling for apps, emulators, and PSX packs.
* Removes unnecessary wrapper folders while preserving the correct EBOOT folder.
* Includes `config.txt` and clean header-only starter catalogs.
* The release package contains only the `PKGi Enhanced` application folder; no separate README file is included.

## Installation

Extract the release and copy `PKGi Enhanced` to `ms0:/PSP/GAME/`.
