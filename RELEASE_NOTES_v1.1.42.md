# PKGi Enhanced v1.1.42

## Download

[Download PKGi Enhanced v1.1.42](http://archive.org/download/pkgi-enhanced/pkgi-enhanced-v1.1.42.zip)

## Changes

* Fixed PSX ZIP extraction when a ZIP already contains a single game folder.
* Prevents doubled paths like `game/game/EBOOT.PBP`.
* Keeps PSX ZIP packs extracting cleanly into the folder selected by the user.

## Install

1. Download and extract the release ZIP.
2. Copy the `PKGi Enhanced` folder to:

```text
ms0:/PSP/GAME/
```

3. Launch PKGi Enhanced from the PSP Game menu.

## Updates

PKGi Enhanced includes a built-in updater for future releases.
