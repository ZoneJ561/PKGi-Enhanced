# PKGi Enhanced v1.1.43

## What's New

* Added a built-in **Update catalogs** option to the PSP app menu.
* Catalog files can now be refreshed over Wi-Fi using PSP-compatible HTTP links.
* Catalog downloads are restricted to recognized PKGi Enhanced category filenames.
* Updated catalogs are loaded automatically after the download completes.

## Download

[Download PKGi Enhanced v1.1.43](http://archive.org/download/pkgi-enhanced/pkgi-enhanced-v1.1.43.zip)

Copy the `PKGi Enhanced` folder to `ms0:/PSP/GAME/` or use the app's built-in updater when upgrading from an earlier release.
